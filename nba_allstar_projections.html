Project Overview

This project involved analyzing historical NBA data to predict the 24 players to be selected for the 2023 NBA All-Star team. The analysis utilized various statistical modeling techniques, including regression, classification, and hyperparameter tuning, to determine the top players most likely to be chosen for the All-Star game.

Features

  • Data Analysis: Leveraged historical player performance metrics and team success data to inform predictions.
  • Statistical Modeling: Applied regression, classification, and hyperparameter tuning techniques to create predictive models.
  • Model Evaluation: Evaluated the model's performance, achieving a prediction accuracy of 70.8% for the All-Star selections.

Technologies Used

  • Python: Used for data analysis, statistical modeling, and prediction.
  • Libraries: Pandas, Scikit-Learn, and other Python libraries for data preprocessing, modeling, and evaluation.
