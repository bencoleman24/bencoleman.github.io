Project Overview

This tool efficiently processes and analyzes customer reviews from Amazon products. It leverages state-of-the-art sentiment analysis and data processing techniques to parse through vast amounts of text, extracting the most pertinent information and compiling it into an easily digestible format. Amazon now has a tool similar to this where they provide a summary of what the reviews say on the product page. At the time I did the project I don't believe they did.

How It Works

Inputting an Amazon product link into the Python script initiates a process of fetching and analyzing customer reviews, which then creates an informative and detailed report. This report showcases key insights, sentiment scores, and trends drawn from the review data.

Features

This project features advanced sentiment analysis, utilizing NLP techniques to determine the sentiment of user reviews and providing key takeaways along with numeric sentiment score values. It includes insightful data visualization, generating clear and intuitive charts and graphs to represent the data, making overall sentiment trends easy to understand. Additionally, it offers comprehensive reporting by producing detailed PDF reports that summarize findings, complete with visual aids and key takeaways.

Usage

To generate a report, follow these steps:

  1. Set up your .env file in the project root with your RapidAPI and OpenAI API keys.
  2. Run the script.
  3. Enter the Amazon product URL when prompted.
  4. Enjoy the PDF report, which will be saved in the working directory.

Limitations and Scope

There are several limitations to this prototype tool:

  • Due to rate limits for the API, it currently processes up to 10 pages of reviews (modifiable).
  • Specifically made for Amazon products, but could be adapted for other e-commerce sites with altered API utilization.
  • Works best for products with a substantial number of reviews.
  • Models may inaccurately classify the sentiment of reviews; scores are insightful but not consistently accurate.

    • Note: This is a prototype and may not be consistently reproducible due to certain limitations and dependencies.

Tools and Technologies

Report Generation

For report generation, Matplotlib and Seaborn were used to create charts and graphs for visualization. ReportLab was employed to generate the final PDF report, enabling complex layouts and the embedding of visualizations, statistics, and text.

Analysis

The analysis utilized Hugging Face Transformers as the primary tool for sentiment analysis, known for their state-of-the-art performance in NLP tasks. TextBlob served as a supplementary tool for aggregate sentiment analysis, providing a polarity score to represent the overall sentiment tone of a text. Additionally, the OpenAI API was leveraged for other NLP tasks, such as summarizing key points from reviews.
