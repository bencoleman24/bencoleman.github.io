Project Overview

Green Space Explorer is a dynamic social platform that allows users to discover and share insights on green spaces like parks and trails. This platform leverages the Google Maps API to provide an intuitive way for users to find nearby parks. I worked specifically on the backend and machine learning aspects of the project.

Features

  • Interactive Map: Utilizes the Google Maps API to locate and explore green spaces in your vicinity.
  • User Ratings and Posts: Users can share experiences and rate green spaces, helping others discover new locations.
  • Secure User Management: Safeguards user credentials using advanced hashing techniques before storage in our MySQL database.
  • Dynamic Content Sharing: Users can post comments and rate parks.
  • Machine Learning: Developed two AI components: a content moderation system using a pre-trained model, and a park scoring feature incorporating sentiment analysis from user comments.

Technologies Used

Frontend: React.js with Bootstrap and CSS for a polished, responsive user interface (developed by the frontend team).

Backend: Python and Flask were used to build the API, which manages user interactions, database access, and external APIs.

Databases: User data, posts, and ratings are stored securely in MySQL and MongoDB databases hosted on Microsoft Azure.

Machine Learning: Implemented content moderation and sentiment analysis for calculating park scores based on user comments.

Cloud:

  • Flask API Deployment: Deployed to an Azure Virtual Machine, using nginx to handle requests and gunicorn for concurrent processing.
  • Azure Cosmos DB: Used for NoSQL MongoDB data storage, supporting efficient querying based on "placeIds" to manage user interactions.
  • Azure MySQL: Structured SQL solution for securely storing user authentication information.
  • Azure App Service: Hosted the React frontend for a reliable, responsive user experience.
