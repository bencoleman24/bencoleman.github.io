Project Overview

This project was conducted as part of the Econometrics for Big Data course at UW-Madison in Spring 2023. The goal was to statistically analyze climate change trends, exploring temperature patterns and emissions data over the past 50 years. The project utilized various econometric and statistical techniques to understand the impact and implications of climate change.

See the full project details in 'Data Analysis and Code.pdf', which includes a summary of findings, detailed analysis, and R code outputs.

Key Analyses

  • Temperature Trends: Analyzed historical temperature data to detect long-term patterns and anomalies.
  • Emissions Data: Examined the relationship between emissions and temperature using econometric techniques.
  • Forecasting: Utilized Seasonal ARIMA, Multiple Linear Regression, Random Forests, and Lasso Regression for forecasting temperature and emissions trends.
  • Model Evaluation: Compared model performance to assess the best predictive methods for climate-related data.

Technologies Used

  • R: Utilized for data management, statistical analysis, and generating visualizations.
  • Statistical Modeling: Applied methods including Time Series Analysis (ARIMA), Multiple Linear Regression, Random Forests, and Lasso Regression.
